# V1 路线图

## 版本愿景

V1 不是“功能堆砌版阅读器”，而是一个具备以下特征的成熟桌面客户端：
- 阅读体验稳定
- 多格式导入可用
- AI 朗读可靠
- 模型/音色切换直观
- 对成本、错误与性能有清晰反馈

## 路线图拆解

### V0.1
- 项目骨架
- 文档体系
- txt 阅读闭环

### V0.2
- TTS provider 接入
- 音色选择
- 播放控制

### V0.3
- 设置页完善
- 本地持久化
- 最近阅读与书架

### V0.4
- epub / md 支持
- PDF 能力评估
- 阅读体验优化

### V1.0
- AI 摘要
- 名词解释
- 阅读辅助面板
- 更完善的异常与成本提示

## 发布策略建议

- 内测阶段以 macOS 优先
- 稳定后扩展 Windows
- Linux 作为开发者友好支持平台
